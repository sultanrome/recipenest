# RecipeNest

A full stack recipe web application where users can discover, 
share and manage recipes online.

## Tech Stack

### Frontend
- React.js
- HTML / CSS
- JavaScript

### Backend
- ASP.NET Core Web API
- C#
- SQL Database

## Features
- Browse and search recipes
- User authentication and login
- Add and manage your own recipes
- Responsive design for mobile and desktop

## Project Structure
- `/recipenest` — React.js frontend
- `/recipenest-api` — ASP.NET Core Web API backend

## About
Built as part of BSc Computer Science at University of Bedfordshire.
A full stack application demonstrating React frontend connected 
to a .NET REST API backend with database integration.
